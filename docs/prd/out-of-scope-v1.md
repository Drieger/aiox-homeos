# Out of Scope (v1)

**Módulo Books (removido em v1.1 — candidato a v2+):**
- CRUD de livros (título, autor, anos, status, imagem de capa)
- Filtros e busca na lista de livros
- Página de detalhe do livro com sinopse editável inline
- Editor de notas rich text (TipTap) por livro com dirty state
- Upload de imagem de capa (JPEG/PNG/WebP)
- Integração com APIs externas (Open Library, Google Books)
- Exportação de dados de livros (CSV, JSON)

**Infraestrutura:**
- Multi-usuário ou autenticação
- Cloud sync ou backup remoto
- Mobile-first ou app nativo
- Redimensionamento automático de imagens no servidor
- Outros módulos além de Home e Design System (filmes, música, finanças — futuras epics)
