## Story

**As a** usuário do HomeOS,
**I want** navegar pelo módulo KB, ver meus documentos e criar/excluir documentos,
**so that** eu possa gerenciar meu acervo de conhecimento.

## Complexity

**Estimativa:** M (Medium) — layout 3 colunas, CRUD de documentos, caderno padrão, Server Actions.

## Scope

**IN:**
- Layout 3 colunas em `/kb`: sidebar KB | lista de documentos | área de conteúdo (placeholder)
- Sidebar KB com lista de cadernos e contagem de documentos por caderno
- Lista de documentos ordenada por `updatedAt` desc: título, caderno, data de modificação
- Server Action: criar documento (título "Sem título") com redirect para `/kb/[id]`
- Server Action: excluir documento (com redirect para `/kb` após exclusão)
- Dialog de confirmação para exclusão (usando `shadcn/ui Dialog`)
- Garantir caderno padrão "Geral" na primeira execução (seed na Server Action de criação se notebooks vazio)
- Testes unitários e de integração das Server Actions
- Testes de componente para o layout e lista de documentos

**OUT:**
- Editor Markdown (Story 3.3)
- Organização por tags (Story 3.4)
- Wiki-links (Story 3.5)
- Busca full-text (Story 3.6)
- Conteúdo real na área de conteúdo (apenas placeholder "Selecione um documento")

## Acceptance Criteria

1. Layout 3 colunas implementado em `/kb`: sidebar KB (cadernos) | lista de documentos | área de conteúdo — todos visíveis simultaneamente em desktop (≥ 1280px)
2. Sidebar KB exibe lista de todos os cadernos com contagem de documentos de cada um
3. Lista de documentos exibe: título, nome do caderno, data de modificação — ordenada por `updatedAt` desc
4. Botão "Novo Documento" cria documento com título "Sem título" no caderno "Geral" e redireciona para `/kb/[id]`
5. Botão de exclusão abre Dialog de confirmação; ao confirmar, remove o documento e redireciona para `/kb`
6. Caderno padrão "Geral" existe sempre: criado automaticamente se não houver nenhum caderno ao criar o primeiro documento

### Padrão de Server Actions

```typescript
// src/lib/kb/actions.ts  ← criar novo arquivo
"use server";
import { db } from "@/lib/db";
import { notebooks, documents } from "@/lib/db/schema";
import { ulid } from "ulidx";
import { eq, desc, count } from "drizzle-orm";
import { revalidatePath } from "next/cache";
import { redirect } from "next/navigation";

export async function createDocument() {
  // Seed caderno "Geral" se não existir
  const existing = await db.select().from(notebooks).limit(1);
  let notebookId: string;
  if (existing.length === 0) {
    const id = ulid();
    await db.insert(notebooks).values({ id, name: "Geral" });
    notebookId = id;
  } else {
    notebookId = existing[0].id;
  }
  const id = ulid();
  await db.insert(documents).values({ id, notebookId, title: "Sem título", content: "" });
  redirect(`/kb/${id}`);
}

export async function deleteDocument(id: string) {
  await db.delete(documents).where(eq(documents.id, id));
  revalidatePath("/kb");
  redirect("/kb");
}
```

```typescript
// src/app/(app)/kb/page.tsx
import { KbSidebar } from "@/components/kb/kb-sidebar";
import { DocumentList } from "@/components/kb/document-list";
import { getNotebooks, getDocuments } from "@/lib/kb/actions";

export default async function KbPage() {
  const notebooks = await getNotebooks();
  const docs = await getDocuments();
  return (
    <div className="flex h-full overflow-hidden">

      <DocumentList documents={docs} />
      <div className="flex flex-1 items-center justify-center text-muted-foreground">

      </div>
    </div>
  );
}
```
