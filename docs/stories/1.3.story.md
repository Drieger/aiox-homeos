## Story

**As a** usuário,
**I want** navegar entre seções do HomeOS via sidebar persistente,
**so that** todas as features sejam acessíveis a partir de um ponto central de navegação.

## Complexity

**Estimativa:** M (Medium) — Componentes React com state management, rotas Next.js, comportamento responsivo mobile, testes de componente.

## Scope

**IN:**
- `AppSidebar` colapsável com navegação Biblioteca/Livros e Design System
- Route group `(app)` com layout compartilhado
- Redirect `/` → `/books`
- Comportamento mobile com Sheet (hamburger)
- Zustand store mínimo (`sidebarCollapsed`) — versão provisória para esta story
- Testes TDD de componente para AppSidebar

**OUT:**
- Conteúdo real das páginas `/books` e `/design-system` (placeholders apenas)
- Tema claro/escuro (Story 1.4)
- `QueryClientProvider` e `ThemeProvider` (Story 1.4)
- Autenticação ou rotas protegidas
- Página de configurações ou perfil de usuário

### Route Group `(app)`

O diretório `(app)` é um **route group** do Next.js — o parêntese NÃO aparece na URL. Ele permite que `/books` e `/design-system` compartilhem o mesmo layout (com sidebar) sem afetar o root layout.

```
URL /books → src/app/(app)/books/page.tsx
URL /design-system → src/app/(app)/design-system/page.tsx
```

[Source: docs/architecture/source-tree.md#route-group-app]

### Zustand Store — Dependência com Story 1.4

O `AppSidebar` precisa do store `sidebarCollapsed`. Como a Story 1.4 cria o store, esta story deve:
1. Criar um store mínimo provisório OU
2. Implementar diretamente o `ui-store.ts` necessário para o sidebar funcionar

**Recomendação:** Criar `src/store/ui-store.ts` com apenas `sidebarCollapsed` nesta story. A Story 1.4 complementará com `theme` e os providers. Isso evita dependência bloqueante entre stories.

```typescript
// src/store/ui-store.ts — versão mínima para esta story
import { create } from "zustand";
import { persist } from "zustand/middleware";

interface UIStore {
  sidebarCollapsed: boolean;
  setSidebarCollapsed: (collapsed: boolean) => void;
  toggleSidebar: () => void;
}

export const useUIStore = create<UIStore>()(
  persist(
    (set) => ({
      sidebarCollapsed: false,
      setSidebarCollapsed: (collapsed) => set({ sidebarCollapsed: collapsed }),
      toggleSidebar: () =>
        set((state) => ({ sidebarCollapsed: !state.sidebarCollapsed })),
    }),
    { name: "ui-store" }
  )
);
```
